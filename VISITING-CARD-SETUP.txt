🎯 HOW IT WORKS:
1. User clicks "View Visiting Card" button on any destination
2. Modal popup appears with dark overlay
3. Visiting card image is displayed in the center
4. User can close by:
   - Clicking the X button
   - Clicking outside the card (on the dark overlay)
   - Pressing ESC key

✨ FEATURES:
- Smooth fade-in animation
- Zoom-in effect when opening
- Blur background effect
- Responsive design (works on mobile)
- Close button with hover effect
- ESC key support
- Click outside to close

📝 NOTE:
If a visiting card image is not found, the system will show an alert message.
Make sure all image file names match exactly as listed above (case-sensitive).
