🔧 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════

PROBLEM: Website not loading properly
SOLUTION: Make sure all 3 files (HTML, CSS, JS) are in same folder

PROBLEM: Styles not showing
SOLUTION: Check style.css is in same folder as index.html

PROBLEM: Buttons not working
SOLUTION: Check script.js is in same folder as index.html

PROBLEM: Images not loading
SOLUTION: Check internet connection (using online images)

PROBLEM: Mobile menu not working
SOLUTION: Clear browser cache and reload
