### Visual Style:
```
┌─────────────────────────────────────┐
│           [🏔️ Mountain Icon]        │
│      (Animated Pulse Effect)        │
│                                     │
│     Plan Your Dream Trip            │
│  Fill in your details and we'll...  │
│  ═══════════════════════════════    │
└─────────────────────────────────────┘
│                                     │
│  👤 Full Name                       │
│  [Enter your name...............]   │

│  📧 Email    |    📱 Phone          │
│  [email...]  |  [+91 XXXXX...]     │
│                                     │

│                                     │

### Characteristics:
- 🎨 Colorful green gradient (#27ae60)
- ✨ Animated mountain icon (pulse effect)
- 🎭 Fancy animations (scale, fade, transform)
- 🎪 Playful, fun appearance
- 🌈 Gradient backgrounds
- 📐 Smaller width (650px)
- 🎯 Icons in every label
- 🔵 Rounded corners (25px)
- 💫 Backdrop blur effect

## 💼 NEW DESIGN (Corporate & Professional)

### Visual Style:
```
┌─────────────────────────────────────────────┐
│  Book Your Adventure                    ✕   │
│  Complete the form below and our travel...  │
│  ─────────────────────────────────────────  │

│  [                                    ]     │

### Characteristics:
- 💼 Professional blue (#3182ce)
- 🎯 Clean, minimal design
- ⚡ Fast transitions (0.2s only)
- 🏢 Corporate appearance
- ⬜ Solid colors, no gradients
- 📐 Wider layout (900px)
- 📝 Simple text labels
- 🔲 Subtle corners (12px)
- 🎨 No backdrop blur

## 📊 Detailed Comparison Table

| Feature | Old Design | New Design |
|---------|-----------|------------|
| **Width** | 650px | 900px |
| **Color Theme** | Green (#27ae60) | Blue (#3182ce) |
| **Button Style** | Gradient + Shadow | Solid Color |
| **Header** | Gradient background | White with border |
| **Icon** | Animated mountain | None (removed) |
| **Label Icons** | Yes (all fields) | No (clean text) |
| **Border Radius** | 25px (very rounded) | 12px (subtle) |
| **Animations** | Multiple (pulse, scale, fade) | Minimal (fade only) |
| **Backdrop** | Blur effect | Simple overlay |
| **Form Layout** | Mixed (1 & 2 column) | Consistent 2-column |
| **Padding** | 40px | 60px (more spacious) |
| **Typography** | Bold, playful | Clean, professional |
| **Close Button** | Circle with shadow | Simple icon |
| **Submit Text** | "Send Inquiry ✈️" | "Submit Inquiry →" |
| **Security Icon** | Shield | Lock |
| **Overall Feel** | Fun, adventurous | Professional, trustworthy |
