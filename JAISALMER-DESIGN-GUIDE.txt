✨ ANIMATION SHOWCASE
═══════════════════════════════════════════════════════════════════════════════

1. HERO SECTION
   ┌─────────────────────────────────────────────────────────────────────────┐
   │ • Background: Pulsing gradient (8s loop)                                │
   │ • Badge: Floating animation (3s loop)                                   │
   │ • Title: Shimmer effect (3s loop)                                       │
   │ • Content: Fade in from bottom on load                                  │
   │ • Parallax: Moves slower than scroll                                    │
   └─────────────────────────────────────────────────────────────────────────┘

2. HIGHLIGHT CARDS
   ┌─────────────────────────────────────────────────────────────────────────┐
   │ • On Scroll: Fade in + slide up (staggered)                             │
   │ • On Hover: Scale up + lift + shadow increase                           │
   │ • Icon: Pulse ring animation (2s loop)                                  │
   │ • Shine: Light sweep on hover                                           │
   └─────────────────────────────────────────────────────────────────────────┘

3. ITINERARY ITEMS
   ┌─────────────────────────────────────────────────────────────────────────┐
   │ • On Hover: Slide right + border color change                           │
   │ • Icon: Pulse ring + rotation                                           │
   │ • Title: Underline grows from left                                      │
   │ • Left Border: Grows from top on hover                                  │
   └─────────────────────────────────────────────────────────────────────────┘

4. GALLERY ITEMS
   ┌─────────────────────────────────────────────────────────────────────────┐
   │ • On Scroll: Fade in + slide up                                         │
   │ • On Hover: Lift + scale + image zoom + rotate                          │
   │ • Overlay: Slide up from bottom                                         │
   │ • On Click: Lightbox with zoom in effect                                │
   └─────────────────────────────────────────────────────────────────────────┘

5. BUTTONS
   ┌─────────────────────────────────────────────────────────────────────────┐
   │ • On Hover: Lift + scale + shadow increase                              │
   │ • Background: Circular expand from center                               │
   │ • On Click: Ripple effect                                               │
   └─────────────────────────────────────────────────────────────────────────┘
