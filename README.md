## 🚀 How to Use

### Option 1: Direct Opening
1. Download all files to a folder
2. Open `index.html` in any modern web browser
3. That's it! The website is ready to use

### Option 2: Local Server (Recommended)
1. Install [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) in VS Code
2. Right-click on `index.html`
3. Select "Open with Live Server"

### Option 3: Deploy Online
Deploy to any of these platforms:
- **Netlify** - Drag and drop folder
- **Vercel** - Connect GitHub repo
- **GitHub Pages** - Push to GitHub and enable Pages
- **Hostinger/Bluehost** - Upload via FTP

## 🎨 Customization Guide

### Change Colors
Edit the CSS variables in `style.css`:
```css
:root {
    --primary-color: #ff6b35;      /* Main brand color */
    --secondary-color: #004e89;    /* Secondary color */
    --dark-color: #1a1a2e;         /* Dark backgrounds */
}
```

### Update Content
- **Company Name**: Search and replace "The Daydream Trek" in `index.html`
- **Contact Info**: Update phone, email, address in footer section
- **Images**: Replace Unsplash URLs with your own images
- **Destinations**: Edit destination cards in the destinations section
- **Packages**: Modify package details and pricing

### Add WhatsApp Number
In `script.js`, line 185:
```javascript
const phoneNumber = '919876543210'; // Replace with your number
```

### Modify Sections
- Each section has a clear comment in `index.html`
- Simply edit the content within each section
- Maintain the class structure for styling
